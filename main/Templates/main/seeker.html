Greetings seeker! You are creating a map to find buried treasure. First design your map, then choose a search algoirthm to find your treasure!

  • Run


    Click run to apply the selected search algorithm on the current treasure map.

  • Select Algorithm


    Choose an algorithm from the dropdown menu to select a search algorithm to use. Then hit run to find a path!

  • Select Change


    Choose how you want to edit the map. Select the type of object you want to create and each mouse click on a tile will turn the tile to the selected object.
