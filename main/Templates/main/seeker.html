Seeker!

Seeker is a game where you can use different search algorithms to find the GOLDEN treasure.

Click here for instructions.

Greetings seeker! You are creating a map to find buried treasure. First design your map, then choose a search algoirthm to find your treasure!

  • Play: Run applies the selected search algorithm on the current treasure map. Reset turns all the visited tiles to unvisited. Reset All undos all changes made to the map.

  • Size: Use this to either add columns or add rows to the map.

  • Select Algorithm: Choose an algorithm from the dropdown menu to select a search algorithm to use. Then hit Run to find a path!

  • Select Change: Choose how you want to edit the map. Once you select a change, any tile you click will become a tile of your chosen type. If you select Obstacle, any tile you click will become an Obstacle tile.
